git
***[[Инициализация гит-системы - git init]]*** - инициализация репозитория (используется когда нужно инициализировать новый репозиторий в Git системе)

***[[Добавить - git add]]*** - добавить в индекс (промежуточное состояние) файл для коммита (перед коммитом нужно добавить файлы которые система Git должна отследить)

***[[Коммит - git commit]] -m "name_file*** - создать снимок (коммит) 

***[[Статус репозитория - git status]]*** - проверить состояние репозитория 
*Пример вывода:*
	[[Статус репозитория - git status]] -s 
	M README.md
	A test.txt
	?? config.json
*Где:*
	М - модификация
	А - добавлен в индекс
	?? - неотслеживается

***[[Информация по индексу - git diff]]*** - посмотреть что изменилось в репозитории с последнего коммита

***[[Логи репозитория - git log]]*** - посмотреть логи коммитов

***[[Конфигурации - git config]]*** - установить конфигурации репозитория
*Например:*
	***[[Конфигурации - git config]] --global user.name*** - установить глобальное имя пользователя

***[[Удаление из индекса - git restore]]*** - восстановить файл до состояния нового коммита

***[[Откатить - git checkout]]*** - откатить изменения до последнего коммита

***[[Удалить - git clean]] ==-n***== - проверить что будет удалено (удаляет файлы и директории)

***[[.gitkeep]]*** - такой файл указывается в пустой директории чтобы система гит могла инициализировать директорию

***[[Ветка - git branch]]*** - создать новую ветку

***[[Переключение на ветку - git switch]] -c*** - создать новую ветку и сразу переключиться на неё (либо без флага **-с** если требуется просто переключиться)

***[[Временная фиксация изменений - git stash]]*** - временно зафиксировать изменения 

***[[Слить ветки - git merge]]*** - слить ветку

***[[Отправить в репозиторий - git push]]*** - отправить коммит в удаленный репозиторий

***[[Отменить слияние веток - git reset]]*** - для отмены слияния если коммит ещё не опубликован

***[[Безопасная отмена слияния - git revert]]*** - для безопасной отмены опубликованного коммита

***[[Визуальный инструмент разрешение конфликтов - git mergetool]]

***[[Перетащить коммит на другую ветку - git rebase]]
