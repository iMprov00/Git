
***[[Добавить - git add]] .*** - добавить все файлы

***[[Добавить - git add]] -A*** - добавить все изменения

***[[Добавить - git add]] -p file.name*** - частично добавить изменения
*Далее попросит выбрать варианты:*
	 **y** - добавить изменения
	 **n** - пропустить
	 **s** - разбить изменения на более мелкие части
	 **e** - ручное редактирование изменений
