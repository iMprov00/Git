
***[[git branch]] -v*** - показывает последний коммит каждой ветки

***[[git branch]] --merged*** (*можно ещё* **--no-merged**)- показывающие ветки относительно их статуса слияния с текущей веткой

***[[git branch]] -a*** - просмотр удаленных веток

***[[git branch]] -d*** - удаление веток

***[[git branch]] -D*** - принудительно удалить ветку
